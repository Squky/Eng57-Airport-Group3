# Eng57-Airport-Group3

The aim of this project is to develop an application for an airport.

## Installation 
1. Download zip file from repo from 
`https://github.com/Squky/Eng57-Airport-Group3`

2. Extract all files from zip

3. Open and run, run file on Ide


## Class setup
Airport_class - flight and passenger instances are stored in the attributes of this class, In the form of a flight list and wait list. 

Aircraft_class - Is parent class. This class holds the attributes for the different aircraft's. 

Planes_class and Helicopter_class -  are sub classes, that inherit from Aircraft_class, with there own attributes.

People_class - This is parent, that holds attributes for people. 

Passenger_class - This is a sub class that inherits form People_class. This class is for passanger details, and has a method for print out passengers details.

Staff_class - This is a sub class, that also inherits from People_class.
